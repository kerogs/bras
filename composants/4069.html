4069 ​

WARNING

Le composant utilisé lors des montages est le mc14069ubcp de chez MOTOROLA (DATASHEET).

Description ​

Le 4069 est un composant qui comporte 6 portes logiques inversées.

Obligatoire

Le 4069 est obligatoire et doit être situé entre le HMI et l'Arduino.

L'utilisation du 4069 est obligatoire puisque le protocole de communication entre le HMI et l'Arduino n'est pas le même.

  • HMI : UART RS-232
  • Arduino : UART TTL
