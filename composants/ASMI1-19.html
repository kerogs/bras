ASMI1-19 ​

WARNING

Le Shield n'est pas disponible sur internet. C'est un shield conçu spécialement pour l'arduino et le projet B.R.A.S. Il est également compatible pour être utilisé en dessous du DRI0023

Description ​

Ce shield permet de simplifier les connections entre les différents éléments du B.R.A.S. Il est composé également de composants nécessaire au bon fonctionnement du projet.

Le ASMI1 fonctionne sur une tension VIN de 12V. Il possède un régulateur de tension afin de la convertir de 12V -> 9V afin d'être compatible pour l'utilisation du csn-a2

Composant utilisé ​

  • 4069
  • Régulateur 12v -> 9v
